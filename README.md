## üì• Installation & Setup

1.  **Clone the Repository**
    ```bash
    git clone [https://github.com/Samir-Singh900/echo-ai.git](https://github.com/Samir-Singh900/echo-ai.git)
    cd echo-ai
    ```

2.  **Install Dependencies**
    ```bash
    pip install flask flask-cors groq
    ```

3.  **Configure API Key**
    * Open `app.py`.
    * Replace the placeholder `API_KEY` with your Groq Cloud API key.

4.  **Run the Application**
    ```bash
    python app.py
    ```
